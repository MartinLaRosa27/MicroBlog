## MicroBlog

### Descripción:

Aplicación web donde un usuario registrado puede registrar diversos posts, a los cuales podrá acceder.

### Tecnologías:

HTML / CSS / SCSS / JavaScript / TypeScript / ReactJS / Python / Flask / MySQL
